How to verify the rules:
  • One tile per step: Each iteration collapses exactly ONE cell (changes from grey to colored)
  • Watch for patterns: As tiles collapse, check if adjacent tiles connect properly
  • Blank tiles: Should only connect to matching directional tiles
  • Directional tiles: Lines should flow naturally between neighbors
  • Constraint propagation: After a cell collapses, neighbors' options may be reduced
  • No contradictions: If you see mismatched connections, rules need adjustment
